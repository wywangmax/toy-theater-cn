# 玩具学堂

一个参考 Toy Theater 游戏墙体验、但使用全新中文品牌和内容的儿童互动学习网站。当前定位是给小学阶段孩子、家长和老师使用的“打开即玩”中文互动游戏入口。

## 当前包含

- 41 个可打开的互动游戏，覆盖数学、语文、英语、艺术、音乐、益智、游戏和教师工具
- 学科、年级、孩子/老师模式筛选
- 搜索、推荐游戏、课堂常用工具入口
- 统一试玩台，所有活动都能在页面内直接打开
- 老师模式下显示课堂使用建议

## 内容方向

- 数学：数感、口算、百数表、分数、时钟、人民币找零、图形和数轴
- 语文：拼音、识字、词语配对、成语、古诗、量词和看图组句
- 英语：字母、大小写配对、首音、CVC 拼词、高频词、颜色数字和简单句
- 艺术 / 音乐 / 益智 / 游戏：画板、颜色混合、小钢琴、节奏、图形记忆、迷宫、逻辑积木和小游戏挑战
- 教师工具：计时器、转盘、随机点名、小组计分、课堂卡片和白板

## 打开方式

直接在浏览器打开 `index.html` 即可。
